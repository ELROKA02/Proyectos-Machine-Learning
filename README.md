# Proyectos de Machine Learning

Repositorio base que centraliza proyectos de Machine Learning desarrollados en Python. Cada proyecto está organizado de forma independiente con su propia estructura, entorno virtual y dependencias.

## Estructura del repositorio

```

├── python/          # Proyectos desarrollados en Python
└── datasets/        # Datos compartidos entre proyectos
```
